線上直譯器：Repl.it

線上直譯器：Repl.it

使用 Python 程式語言所寫出的程式碼，不需要透過編譯器（compiler）將它編譯成可執行檔案，僅需要 Python 直譯器（interpreter）一行一行翻譯就可以執行。

Repl.it

Repl.it是個小而美的雲端整合式開發網站。

使用者只要有瀏覽器與網路連線，就能直接寫程式碼、執行、除錯，不需在本機安裝任何軟體！

而repl.it支援Google、Facebook等帳號登入，除了能有效的管理撰寫的程式碼之外，在剛開始學習程式時不失為一個快速上手的好網站，並且免去了在電腦上安裝直譯器與開發工具的麻煩。

  1. 支援多種帳號登入：
    Google、Facebook、GitHub或申請repl.it的帳號

  2. 紀錄所有的程式碼檔：
    可以從右上方的my repls找到編寫過的程式碼檔。

  3. 支援多種語言：
    點選New Repl，可以開啟一個新的程式碼檔，但要先選擇要使用的程式語言。
    我們選擇Python。

  4. 介面介紹：
    大致上分成三個工作區塊，
    ● 藍色部分：點選齒輪開啟網站介面設定，可以更改網站排版、工作區顏色、字體大小等視覺化的設定
    ● 紅色部分：程式碼工作區，會依照選擇的程式語言改變關鍵字的顏色、智慧的完成句子，幫助使用者撰寫程式碼。
    ● 綠色部分：Python直譯器，執行的結果會顯示在這裡喔！

  5. 功能按鈕介紹：
    ● 紅色部分：檔案名稱，點選鉛筆可做修改
    ● 綠色部分：分享→快速分享網址，讓其他人藉由瀏覽器觀看程式碼。執行→運行程式碼
    ● 深藍部分：下載，可以下載正在撰寫的程式碼檔
    ● 淺藍部分：擴充模組→能選擇repl.it支援的python模組。除錯功能→能進入debug模式，找出程式碼的錯誤之處。
