🗒️【函式Function】和【方法Method】
  • 我們通常將一個常利用到的小功能獨立寫成一個「函式」，並通過名字來調用函式，使程式整體的長度變短。
    • 函式，例如：print("abc" )
  • 方法是一個被定義在類別裡的函式，以物件本身為預設參數，通過名字來調用方法。
    • 方法，例如：list1.append( 6 )
🗒️字串函示與方法(點我查看官方文件)
1️⃣函式
名稱 用法
len( X ) 回傳字串 x 的長度。
max( X ) 回傳字串 x 中，編碼最大的字元。
min( X ) 回傳字串 x 中，編碼最小的字元。
str( X ) 將參數 x 轉換成字串型別。
chr( X ) 傳回參數 x 所表示的Unicode字元。
ord( X ) 傳回參數 x 的Unicode編碼。
2️⃣字串內建的搜尋方法
名稱 用法
find("x") 回傳參數x出現在字串中的最小索引值，沒有則回傳-1
rfind("x") 回傳參數x出現在字串中的最大索引值，沒有則回傳-1
startswith("x") 判斷字串是否以參數x開頭
endswith("x") 判斷字串是否以參數x結尾
count("x") 回傳參數x在字串中出現的次數
3️⃣字串內建的測試方法
名稱 用法
isdigit( ) 判斷字串是否都是阿拉伯數字組成
isalpha( ) 判斷字串是否都是英文字母組成
isalnum( ) 判斷字串是否以數字與英文字母組成
isupper( ) 判斷字串是否都是大寫字母
islower( ) 判斷字串是否都是小寫字母
isspace( ) 判斷字串是否都是空白("")
4️⃣字串內建的轉換方法
名稱 用法
upper( ) 將字串內的字元轉換成大寫，並回傳
lower( ) 將字串內的字元轉換成小寫，並回傳
capitalize( ) 將字串內第一個字元轉換成大寫，並回傳
swapcase( ) 將字串內字元大小寫互轉，並回傳
title( ) 將字串內每個單字的第一個字元轉換成大寫，並回傳
replace(old,new) 將字串內的old參數，取代成new參數，並回傳
5️⃣字串片段運算子
字串變數[ 起始值 : 結束值 : 間隔值 ]
  • 和串列不同，字串有順序且不可變動
  • 一字串的索引值，從前端開始為0
  • 一字串的索引值，從尾端開始為-1
  • 印出指定的索引範圍時，不包含結束值
  • 間隔值預設為1，非必要時可不輸入
索引 0 1 2 3 4 5
內容 P Y T H O N
索引 -6 -5 -4 -3 -2 -1
