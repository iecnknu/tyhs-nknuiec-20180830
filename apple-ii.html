Apple II

Apple II

Apple II是蘋果公司製作的第一部在商業上取得成功的個人電腦，於1977年發布後一共售出了5-600萬台。
在1980－1990年代，Apple II是美國教育系統實際採用的標準電腦。
直至今日，仍有一些還在教室里並能正常使用，Apple II在商業、家庭與學校用戶之間很普及。
Apple II有彩色、高解析度的圖形顯示模式，音效功能與兩種基於BASIC的內建程式語言－Applesoft與Integer與其他微電腦相比，這些功能被完善記載且易於學習。Apple II開啟了個人電腦革命，因為它面向大眾，而不僅僅是科技狂人與工程師。其上市發表與後來的普及更深刻影響了後繼的許多種微電腦。

Apple Il emulator：

為了讓大家體驗1970年代的電腦介面，
找了一款Apple II 的線上模擬器，讓大家試試輸入程式以及下一些指令。
現在預設載入ROM是最後一版的DOS【Apple DOS 3.3】。

https://www.scullinsteel.com/apple2/#dos33master

使用Apple II

與現在的電腦不同的是，要讓電腦動作，
只能透過輸入【指令名稱】來呼叫功能指令，與現在便捷的使用者介面跟滑鼠點擊相去甚遠。

程式目錄

在輸入指令的位置，輸入

CATALOG

按下Enter鍵後便會呼叫出程式目錄。
為了測試蘋果電腦的16位元色彩，我們來執行COLOR DEMO程式【顏色示範】吧！

在指令的位置，輸入

RUN COLOR DEMO
#備註：RUN表示執行，COLOR DEMO就是程式的名稱

總共有5個功能可以選擇，都是透過輸入數字來選擇想要執行的功能。

我們選擇3，KALEIDOSCOPE【萬花筒】。

選擇4，SKETCHING SCREEN【在螢幕上素描】。

點擊滑鼠左鍵開始畫圖，右鍵切換顏色！

用Apple II寫程式

來寫一個自動增值的程式吧

1 X = X + 1
2 PRINT X
3 GOTO 1
RUN

第1行：宣告變數X的值是X+1，變數X預設為0
第2行：印出X
第3行：回到第1行
第4行：RUN【執行程式】
