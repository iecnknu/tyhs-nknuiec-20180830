if抉擇結構

「接下來，我們要來學習能判斷的咒語，if，如此一來，當我們才能把精心設計的流程圖轉換成能運作的魔法！」

  • if敘述
  • if else敘述
  • if elif else敘述
  • 巢狀判斷敘述
🗒️if敘述
  • 日常生活中會依據不同情況做出相對應的選擇。
  • if用法會先執行判斷式，如果符合條件才會進入執行主體。
  • 使用if關鍵字，並須接上一段布林運算式作為判斷式。並以冒號將之與執行主體區隔。
  • 執行主體則需以if關鍵字為基準，以一個空白或tab鍵向右縮排(兩者不可混用)。
if 判斷式(布林運算式) ：
    執行主體

🗒️if else敘述
  • 除了符合條件，也可加進不符合條件時，想要執行的部分。
  • 判斷式不成立的話，就會進入else的執行範圍。
if 判斷式(布林運算式) ：
    執行主體-1
else：
    執行主體-2

🗒️if elif else敘述
  • 多向選擇性，一次處理多個判斷式。
  • elif 為else if 的縮寫。
  • 一個判斷程式可以有0個或多個elif，卻只能有0個或 1 個else。
if 判斷式-1(布林運算式) ：
    執行主體-1

elif 判斷式-2(布林運算式) ：
    執行主體-2

else：
    執行主體-3

🗒️巢狀判斷敘述
  • 巢狀一詞，指得是利用縮排的技巧，讓程式碼內縮排列，看似鳥巢(nest)。
  • 【巢狀if 】為if 敘述裡含有其他 if 敘述，數量不限，但建議不宜過多、影響程式閱讀。

        執行主體-1.1

        if 判斷式-1.1.1(布林運算式) ：
            執行主體-1.1.1

    elif 判斷式-1.2(布林運算式) ：
        執行主體-1.2

elif 判斷式-2(布林運算式) ：

    if 判斷式-2.1(布林運算式) ：
        執行主體-2.1

    else：
        執行主體-2.2
