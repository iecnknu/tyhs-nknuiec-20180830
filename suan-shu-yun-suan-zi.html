算術運算子

算術運算子

  • 表格由高至低排列運算子的優先順序。
  • 在同一個運算式中，優先權越高的會先執行，類似先乘除後加減的概念。
  • 名詞解釋：
運算子 名稱 用法
** 指數運算子 計算運算元的指數結果
// 整數除法運算子 將除法結果取整數
% 模數運算子 計算兩運算元之餘數
* 乘法運算子 計算兩運算元之乘法
/ 除法運算子 計算兩運算元之除法
+ 加法運算子 計算兩運算元之加法
- 減法運算子 計算兩運算元之減法

#  +  使用加法運算子，計算 2加上2

👨‍💻課堂練習

  1. 水沙漏 點我前往

    提示：
使用球缺體積公式，計算出還要多少年才能滴滿水沙漏？
每天滴下100毫升的水

1.名詞解釋：
  π是圓周率(約3.14)   h是球缺的高   𝑅是球體的半徑

2.單位轉換：
  1立方公尺 = 1000公升  1公升 = 1000毫升

3.球體數據：
  球缺的高 : 3.8公尺
  積水高 : 0.2公尺
  球體半徑：2公尺

4.球缺體積公式為：
  π x h x h x (R-h/3)

  2. 幾年幾天

    延續上題，請在最後印出
”需要________年又________天才能滴滿”

  3. 網路傳輸速度

    網路傳輸速度1M = 128KB/秒，檔案大小1MB = 1024KB
在沒有干擾的情況下，以100M的網路速度，全速下載一個256MB的檔案，需要花多少秒？

    電腦的最小單位為 Bit (位元)，以下是所有數量單位的說明與其英文全名：
    1 Byte(位元組) = 8 Bits
    1 Kilobyte (KB) = 1024 Bytes
    1 Megabyte (MB) = 1024 KB
    1 Gigabyte (GB) = 1024 MB
    1 Terabyte (TB) = 1024 GB
    1 Petabyte (PB) = 1024 TB
    1 Exabyte (EB) = 1024 PB
    1 Zettabyte (ZB) = 1024 EB
    1 Yottabyte (YB) = 1024 ZB
