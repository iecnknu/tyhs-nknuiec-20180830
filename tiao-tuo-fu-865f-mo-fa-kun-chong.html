跳脫符號

跳脫符號

有許多無法顯示在螢幕上的符號，使用反斜線或其他功能鍵便能顯示出來，供使用者靈活使用。
這些符號稱之為【跳脫符號】。

🗒️常用符號列表
名稱 招喚鍵
縮排 Tab鍵
註解 #
空白 空白鍵
跳行 \n
跳位 \t
反斜線(跳脫字元) \
🐞縮排 點我前往
縮排 Tab鍵 實現縮排，幫助運行和閱讀程式碼。 
if(True): 
print('Hello')
🐞註解 點我前往
註解 #鍵，shift+數字3 實現備註，幫助紀錄程式碼的用途。 
print('Hello')
上面這段會印出Hello
🐞空白 點我前往
空白 空白鍵 增加適當空白，提高程式碼的閱讀性。 
153*4867+12354-45135/150
#health*life-level/coin
🐞跳位 點我前往
跳位 \t ，反斜線 + t 實現跳位排列，讓程式碼輸出的結果能整齊排列。 
print('普羅14歲' )
🐞跳行 點我前往
跳行 \n ，反斜線 + n 實現跳行排列，讓程式碼輸出的結果能換行顯示。 
print('普羅14歲' )
🐞反斜線(跳脫字元) 點我前往
反斜線(跳脫字元) \，反斜線 可以出單引號、雙引號和反斜線的跳脫符號。 
print('我說:'Python魔法萬歲'')
🗒️補充

在編碼的章節中有提到：
Unicode在Python中的表示法為\u 後面加上四個十六進位的數字。
ASCII在Python中的表示法為 後面加上三個八進位的數字。

也是屬於跳脫符號的一種喔！
