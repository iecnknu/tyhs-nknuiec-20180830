什麼是變數？

🗒️變數規則
變數名稱 = 內容值

在數學裡，等號代表的是等號兩端的數值相等。

在程式中，等號用來把右邊的【內容值】，賦予左邊的【變數名稱】。

variable1 = 'Ghost 1'
variable2 = 123
variable3 = True
variable4 = [1, 'Py']

Python可以一次指派同個值給多個變數，也可以一次指派不同值給多個變數。

Python能依據之後賦予的值改變變數型別。

#同個值給多個變數
variable1 = variable2 = 'Ghost 1'

#不同個值給多個變數
variable3, variable4= 'Ghost 2', 'OneEye'
🗒️變數命名規則
  1. 不能使用保留字
  2. 不能使用特殊符號
  3. 不能以數字開頭
  4. 不能包含空白
  5. 第一個字母必須是中文、英文或者底線【_】，其他字元可由中文、英文、底線或數字組成
  6. 每換一個單字，就改變大小寫，稱為【駝峰式命名】
  7. 使用英文有大小寫的區別
🗒️變數名稱保留字
False None True class def as assert
break continue del elif else if except
finally for global nonlocal raise return try
while with yield pass and not or
is in lambda from import
