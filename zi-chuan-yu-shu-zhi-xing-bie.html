字串與數值型別

字串與數值型別

電腦世界裡的資料，被分成了各種型別。
一個資料的型別會決定該資料的用途、占用的體積空間、顯示的外貌與範圍。

  • 字串型別
  • 數值型別
🗒️字串型別
  • 負責處理文字資料，有著多種表示方式，包含數字、文字、和符號。
  • 字串可以被拆解使用，且具有前後順序的特性。
  • 如果資料要表示為文字，就需要用以下符號裝飾：
符號 名稱 範例
' 單引號 '你是普羅'
" 雙引號 "我是菲絲恩"
''' 三個單引號 '''我是魔法師'''
  • 備註：各種引號切勿混用，以免發生錯誤。
  • 【'''】兩組三個單引號所包圍的語句，可以自由跳行顯示。
🗒️數值型別
  • 任何跟數值相關的資料，都是數值型別。
  • 和字串型別不同的是，數值型別的值可以直接用來做運算。
別稱 名稱 範例
int integer 表示沒有小數點的正負整數，像是1、-5和100。
float float 表示具有小數部分的數字，像是1.54、0.7和-5.4。
bool boolean 為整數的子型別，代表著True、False和None。

🗒️強類型的動態腳本語言

  1. 強類型語言：
    使之強制數據類型定義的語言。
    沒有強制類型轉化前，不允許兩種不同類型的變量相互操作。
    強類型定義語言是類型安全的語言，如Java、C# 和 python。

  2. 弱類型語言：
    數據類型可以被忽略的語言。 與強類型語言相反, 一個變量可以賦不同數據類型的值，允許將一塊內存看做多種類型，比如直接將整型變量與字符變量相加。C/C++、PHP、JavaScript都是弱類型語言。
    JavaScript：

    "2" - 1

  3. 動態類型語言：
    與靜態語言剛好相反，它只在進程運行期間才去進行變數型別檢查。
    不需要給變數指定型別，它會在賦值時，在內部記錄變數型別。

    Python

    str1 = '123'
num1 = 123

  4. 靜態類型語言：
    撰寫程式時要聲明變量的數據類型，而且不同型別的變量不允許直接賦值。
    它的變數類型是編譯期間進行檢查的。

    C語言

    char str1 = '123';
int num1 = 123;
