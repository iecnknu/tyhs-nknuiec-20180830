  • 串列的方法
  • 串列增值-1
  • 串列增值-2
🗒️串列的方法
名稱 用法
append( X ) 在串列list的尾端插入參數X
insert( X , Y ) 在串列list中索引值為參數X的位置插入參數Y
extend( X ) 在串列list的尾端插入參數X中的所有元素，參數X必須為具有順序的值，可以是串列或字串
remove( X ) 刪除串列list中第一個符合參數X的元素
count( X ) 回傳參數X在串列list中出現的次數
clear( ) 將整個list清空
sort( ) 將串列中的元素依照大小排列
reverse( ) 將串列中的元素順序反轉過來
🗒️串列增值-1
  • 串列增值可使用內建的串列函數。

  • append(值)，可在list的最後加入值。
    append，意思是附加。

    list1 = ['a','b']
list1.append('c')

  • insert(位置,值)，可將值插入到指定索引位置上。

    list1 = ['a','b']

# 2 表示插入的位置，'c'則是要插入的值
list1.insert(2,'c')

🗒️串列增值-2
  • 串列增值可使用內建的串列函數。

  • extend( 值 )，只接受有順序的值為參數，也就是有索引值的型別，可以是串列或字串。

    list1 = [1,2]
list2 = [3]

#將list2的內容擴增到list1裡
list1.extend(list2)

  • 以字串為值傳入的話，會將每個字元拆開當作獨立的值，依序加入串列的最後端。
